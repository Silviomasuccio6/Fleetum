# Data Breach Procedure Fleetum

## Flusso operativo

1. Rilevazione.
2. Classificazione.
3. Contenimento.
4. Analisi dati coinvolti.
5. Valutazione rischio.
6. Notifica interna.
7. Notifica al cliente titolare.
8. Valutazione notifica Garante.
9. Eventuale comunicazione interessati.
10. Remediation.
11. Report finale.
12. Registro data breach.
